# Vue-Do List

A live-shared & reactive to-do list built with VueJS & Typescript.

### Project Dependencies

* `Vue 3.0.0`
* `TypeScript 4.1.5`
* `Node v16.5.0`
* `Npm 7.20.1`

### Get Started

* Clone the repo locally and run the project with `npm run serve`
* OR Check out the beta deployment here: `NO LINK YET`
